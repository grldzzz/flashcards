## DIAPOSITIVA 3: INTRODUCCIÓN
- **Contexto**: Necesidad de herramientas de aprendizaje efectivas
- **Problema**: Dificultad en memorizar y retener información clave
- **Solución**: Aplicación móvil con sistema de tarjetas de memoria
- **Valor añadido**: Personalización y seguimiento del aprendizaje

## DIAPOSITIVA 4: OBJETIVOS Y ALCANCE
- Crear una aplicación funcional y completa para Android/iOS
- Implementar sistema de flashcards tradicionales y opción múltiple
- Desarrollar interfaz intuitiva y agradable
- Almacenar datos localmente con persistencia
- Implementar sistema de seguimiento del progreso

## DIAPOSITIVA 5: TECNOLOGÍAS UTILIZADAS
- **Framework**: Flutter 3.7+
- **Lenguaje**: Dart 3.7+
- **Persistencia**: Hive (base de datos NoSQL)
- **Gestión de Estado**: Provider
- **Animaciones**: Flip Card
- **Testing**: Flutter Test, Integration Tests
